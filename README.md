# Audio Editor Compose (НА СТАДИИ РАЗРАБОТКИ)

Простой аудиоредактор на **Kotlin + Jetpack Compose** с поддержкой базового редактирования аудио.

## Возможности

- Загрузка аудиофайлов (wav формата)
- Визуализация аудиоволны
- Вырезать, копировать, вставить фрагменты
- Перемотка и отображение текущего времени
- Экспорт отредактированного файла
